### 可选项

| 参数        | 说明           | 类型  | 可选值  | 默认值  |
| ------------- |:--------:| :-----:|:-----:|:------------:|
| min      | 最小值 | Number | — |  0 |
| max      | 最大值 | Number | — |  100 |
| step      | 步长 | Number | > min and  < max |  1 |
| change      |鼠标离开时候的最终只| Function | — | —  |
| input      |slider改变时候的值| Function | — | —  |
