### 可选项

| 参数        | 说明           | 类型  | 可选值  | 默认值  |
| ------------- |:--------:| :-----:|:-----:|:------------:|
| disabled      | 是否禁用该按钮 | Boolean | `true|false` |  false |
| circle      | 是否是圆角 | Boolean | `true|false` |  false |
| autofocus      | 是否自动地获得焦点 | Boolean | `true|false` |  false |
| type      | 按钮的类型 | String | `button reset submit` |  button |
